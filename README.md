# Open Spotify links in Desktop App

Simple script to open Spotify links directly in the desktop app instead of the web player.

## Installation

### 1. Install a userscript manager

You need one of the following:

- [Tampermonkey](https://www.tampermonkey.net/)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)

### 2. Install the script

Install from [GreasyFork](https://greasyfork.org/en/scripts/573357-open-spotify-links-in-desktop-app)

Or install manually:

1. Create a new script in your userscript manager
2. Copy the content of userscript.js and paste it into your newly created script
3. Save and enable the script
